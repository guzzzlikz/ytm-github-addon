<div style="font-size: 40px">YouTubeMusicDesktop token</div>
<div>
    To get YouTubeMusicDesktop token, go to https://github.com/pear-devs/pear-desktop/releases <br>
    Download latest version <br>
    Then, in app: Plugins -> Api Server -> Enable <br>
    Hostname (best - put in app 0.0.0.0) <br>
    Port <br>
    Authorization strategy - better enable <br>
    After all the preparations, input URL (hostname) and port <br>
    Then, provide an id (any string) <br>
    after that you will receive token
